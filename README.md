# ng2-tutorial [checkout git branches to learn Angular2 app] 

## usage[安装项目]
```
  npm install
  
  npm start

``
## git branches[各个git分支模块说明]
- @NgModule module主模块入门 

> ```git checkout ngModule```

- Component组件化开发 

> ```git checkout component```

- router动态路由  

>```git checkout router```
